# 测试实验文件夹

此文件夹用于存放各种测试实验的代码、数据和结果。

## 文件夹结构

```
experiments/
├── README.md                 # 本文件
├── experiment_1/            # 实验1
│   ├── config.json          # 实验配置
│   ├── run.py               # 实验脚本
│   ├── results/             # 实验结果
│   └── README.md            # 实验说明
├── experiment_2/            # 实验2
│   ├── ...
└── shared/                  # 共享资源
    ├── utils.py             # 共享工具函数
    └── templates/           # 模板文件
```

## 使用说明

1. 每个实验创建一个独立的子文件夹
2. 在实验文件夹中包含：
   - 实验配置文件
   - 实验运行脚本
   - 实验结果数据
   - 实验说明文档

3. 共享资源放在 `shared/` 文件夹中
